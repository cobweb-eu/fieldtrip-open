### Fieldtrip-Open

Fieldtrip-Open is a mobile mapping and data collection platform.

### Software Dependencies

[Install dependencies notes](INSTALL_DEPS.md)

### Install

```
$ fab install_project
$ Provide location of config file > etc/config.example
```

### Deploy Android

```
$ fab deploy_android
```

### Deploy IOS

```
$ fab deploy_ios
```
